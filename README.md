# Lernkarten : German A1 Flashcards
German A1 level wordeck. 

Access here : https://jamadadeharshita.github.io/Lernkarten/

( Word mapping is still in progress)

A minimal, swipeable flashcard app for learning German A1 vocabulary,hosted free on GitHub Pages.

## Features
- 1,362 words merged from *Netzwerk neu A1* and the Goethe-Institut
  "Start Deutsch 1" word list, with real example sentences
- Real spaced repetition (SM-2, the same algorithm Anki uses)
- Swipe to rate  right = know it, left = again, up = easy, down = hard
- Light/dark theme, topic filters, audio pronunciation, progress bar

## Fun fact: the SM-2 algorithm!

SM-2 was written in **1987 by Piotr Woźniak**, a Polish student who
built it to learn English vocabulary himself  by hand, on paper,
before he ever had a computer to run it on. He was chasing an idea from
1885: psychologist **Hermann Ebbinghaus** discovered the "forgetting
curve," showing that memory decays predictably over time unless it's
refreshed at the right moment.

SM-2 turns that into a simple rule: the better you knew a card, the
longer it waits before showing it again. Woźniak's software became
**SuperMemo**  and even though he's since published much more
sophisticated versions (SM-15, SM-18...), SM-2 is still, decades later,
the algorithm quietly running inside **Anki**, the most popular
flashcard app in the world. The version rating your German words above
is functionally the same math a Polish student scribbled in a notebook
in the 80s.

## Fun fact: 

German's famous habit of gluing words together into
endless compounds (*Donaudampfschifffahrtsgesellschaftskapitän*  "Danube
steamship company captain"  is a real, dictionary-adjacent word). The
vocabulary is small; what you *build* with it is not.
